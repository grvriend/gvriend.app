## ğŸš€ Deployment

This site is deployed using GitHub Pages.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/grahamvriend/gvriend.app.git
   ```

2. Open `index.html` in your browser:
   ```bash
   # On Windows
   start index.html
   
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   ```

### Deploy to GitHub Pages

1. Push your changes to the main branch
2. GitHub Pages will automatically deploy the site
3. The site will be available at your configured domain

## ğŸ“ Customization

To customize this portfolio for your own use:

1. **Update Personal Information**:
   - Edit `index.html` to update all text content
   - Replace social media links
   - Update meta tags in the `<head>` section

   - Modify CSS variables in `styles.css` (root section)
   - Adjust gradient colors for orbs

3. **Modify Sections**:
   - Add/remove sections in `index.html`
   - Update navigation links accordingly

4. **Add Images**:
   - Place images in the `img/` directory
   - Update references in HTML
